## Tips
### Change Font
```
sed -i 's/<FontFamily>Tahoma<\/FontFamily>/<FontFamily>Segoe UI Variable Static Small<\/FontFamily>/g' /config/MusicBee/AppData/MusicBee3Settings.ini
```
